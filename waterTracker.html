Project Summary

This project is a blazor web assembly application with an ASP.NET Core web api backend and SQL server DB which I made as part of a programming task for a job interview. The application is used to track a users water intake . The app has the following features:

  • View a list of users and perform CRUD operations on them.
  • View a list of a users water intake records and perform CRUD operations on them
  • Use an API to perform CRUD operations on the users and water intake records

Technologies used:

  • Blazor Web assembly
  • ASP.NET Core web api
  • HTML
  • CSS
  • Entity Framework COre
  • X-unit

Skills used:

  • Performing CRUD operations on a datbase through API Calls
  • Creating views and components and navigating between them
  • Creating and Consuming API services
  • Unit of work and repository pattern
  • Use of solid principles
  • Unit and integration testing
