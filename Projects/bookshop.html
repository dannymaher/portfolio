Project Summary

This project is an e-commerce site which sells books. There are 4 different types of account which can do different things on the site.

Customer

  • View a list of each product and view the details for a specific product such as name, author and price.
  • Add products to their basket where they can see the total cost for each product and a combined total cost of the basket.
  • Order and pay for the items in their basket which is processed through stripe.
  • Update their account information such as change password and user data.

Company

  • The same actions as customer but they have the option to pay up to 30 days after ordering.
  • Can only be created by an admin.

Employee

  • All the actions of customer.
  • View and manage user orders.
  • Update the status of user orders from approved to processing to shipped.
  • Cancel user orders.
  • Can only be created by an admin.

Admin

  • All the actions of employee and customer.
  • Perfrom CRUD operations for categories, products, companies and users.
  • Manage users by changing their roles or locking from logging in.
  • Can only be created by an admin.
