Project Summary

This project is an android application i made as part of a group project in university which allows users to browse through a collection of movies, TV shows and people. The app has the following features:

  • View a collection of trending shows or movies which can be clicked on to navigate to the show/movies details view.
  • View a collection of shows/movies on the discover page which can be filtered by genre, popularity , age rating and release date. Can be clicked on to navigate to the show/movies details view
  • The show/movie details view shows details about it such as the release date, runtime, description, production company, director(s), cast, reviews and similiar and recommended movies/shows.
  • Can leave a review on a show, episode or movie.
  • Can save a show/movie to a list of favourites
  • Can add the release date for a show, movie or episode to their calander to recieve a notification when it will release
  • Can view a person to see information about them as well as the shows and movies they have starred in.
  • Can search for movies, shows and people

Technologies used:

  • Android studio
  • Java
  • XML
  • Firebase DB
  • IMDB API

Skills used:
